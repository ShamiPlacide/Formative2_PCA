# Formative 2 - Principle Component Analysis

**African Banking Crisis PCA**
This project implements Principal Component Analysis from scratch using NumPy to analyse the African Economic, Banking and Systemic Crisis dataset, reducing 11 economic features into principal components while preserving maximum variance.
